Run Buddy

Run Buddy was built using a module from the Coding Boot Camp at The University of Texas at Austin. It utilizes advanced styling for multiple sections to match the provided mock up, including media queries to adapt for various screen sizes. Click the image to explore the site & it's functionality!

Recipe Finder

A team of 3 developers created this website to simlify the process of finding a new recipe. Using buttons, users can search for recipes by either food type or dietary restrictions. They will get a random recipe name, image, ingredient list & link to full recipe, along with the total nutrition facts of the recipe. We used Edamam APIs to get the information & JavaScript including JQuery for the functionality. Additionally, we utilized the Bulma CSS framework for styling.

Bait-and-Tracker

This is an app developed on a team to track the results of fishing competitions. Users can create an account & upload their tournament catches. The site determines winners and displays a leaderboard including photos and timestamps in order to comply with standard angler tournament rules. Non-account holders can also view all the results and individual profiles!
